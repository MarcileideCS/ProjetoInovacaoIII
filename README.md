# ProjetoInovacaoIII

Projeto web sobre Tecnologia construido com HTML, CSS e JavaScript puro.

## Estrutura

- index.html
- styles.css
- script.js

## Como executar

1. Abra a pasta do projeto no VS Code.
2. Abra o arquivo index.html no navegador.
3. Opcional: use a extensao Live Server para atualizar automaticamente.

## O que foi implementado

- Hero com visual moderno e tema tecnologico.
- Radar de tendencias com filtros dinamicos.
- Painel de maturidade com barras animadas.
- Relogio em tempo real e botao de insights.
- Botao de tema surpresa com troca de cores de destaque.
